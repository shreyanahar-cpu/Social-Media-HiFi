# Social-Media-HiFi
A high-fidelity UI/UX design of a social media mobile app created in Figma.

## Project Description

This project is a high-fidelity (Hi-Fi) UI/UX design of a social media mobile application created using Figma. It represents the final visual design with proper colors, typography, images, and realistic user interface elements.

The app focuses on providing a clean, modern, and user-friendly experience where users can interact, create posts, and navigate easily between different sections.

This design demonstrates the complete transformation from low-fidelity wireframes to a fully polished user interface.

## Features

- User Login & Signup  
- Home Feed with posts  
- Profile Screen  
- Create Post functionality  
- Smooth Navigation bar  
- Clean and modern UI design
